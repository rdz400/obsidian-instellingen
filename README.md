# Configuratie Synchronisatie

Deze repository bevat een script om eenvoudig Obsdian-templates en -configuratie te synchroniseren met deze repository.

## Gebruik


1. Check .gitignore
1. Check variabelen in .env (zie .env.template)
1. Voer het synchronisatiescript uit
1. Voor commit, check inhoud van wat wordt gecommit

```bash
./sync-files.sh
```
